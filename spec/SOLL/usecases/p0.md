# P0 Usecases

UC1 Create Template (ADMIN)
- Create form_templates row (owner_group_id must exist)
- Create initial TEST version 1.0.0 (inactive by default)

UC2 Publish TEST (MANAGER)
- Create new version or mark latest TEST as active (P0: mark latest as active)

UC3 Publish PROD (MANAGER)
- Mark chosen PROD version active; ensure only one active PROD

UC4 Start Entity (EDITOR)
- Pick active PROD template_version for template
- Create entities row with status DRAFT and template_version_id locked

UC5 Save Draft (EDITOR)
- Update entities.data_json and updated_at

UC6 Submit (EDITOR)
- status: DRAFT -> SUBMITTED

UC7 Approve/Reject (MANAGER)
- Insert into approvals
- status -> APPROVED_FINAL or REJECTED
- Write audit_log entry
